# Controls & UI Guide

This document explains expected player controls and HUD elements.

Controls (suggested)
- Move: mouse / WASD or arrow keys.
- Aim/Fire: mouse click or space.
- Abilities: number keys or UI buttons.

UI elements
- Scoreboard / leaderboard
- Health bar, level/XP
- Minimap (optional)
- Chat / notifications

Notes: Keep controls responsive. Design for keyboard+mouse primarily, optionally support touch.
